# Online-Offline-Form
Project Introduction – Registration Form

This project is a simple and visually attractive registration form webpage created using HTML and CSS. The main objective of this project is to design a structured and user-friendly form interface with proper alignment and styling.

The webpage contains input fields for name and email, along with radio buttons for gender selection and checkboxes for interests. It also includes a styled submit button with hover effects to enhance interactivity.

CSS is used to create a modern design using gradient backgrounds, rounded corners, box shadows, and smooth hover transitions. Flexbox is applied to center the form both vertically and horizontally on the screen.

This project demonstrates basic form creation, input handling structure, and aesthetic UI design using frontend technologies without any backend functionality.
